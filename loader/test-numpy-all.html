wasthon — numpy suite dashboard

Runs numpy 2.5.1's own test modules (core + numpy.random) against the combined numpy + numpy.random wasm. brython + the VFS + the wasm load once and the modules run sequentially in one interpreter — no per-module reload. Per-module detail goes to the browser console; the page shows the table. Cross-module state is shared (the iframe dashboard stays the isolation reference).
