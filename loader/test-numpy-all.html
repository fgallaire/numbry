wasthon — numpy suite dashboard

Runs modules from numpy 2.5.1's own test suite (numpy/**/tests/test_*.py) against wasthon, each in a fresh page (test-numpy.html?test=…) so no module contaminates another. A minimal pytest shim supplies raises/mark.parametrize/ fixture/… Scores are numpy's own assertions. Only the pytest-portable subset is listed — modules needing numpy.random, C test-extensions, ctypes/mmap/zoneinfo, or hypothesis are out of scope. Failures are real bridge/behaviour gaps (the backlog to fix); a few come from the shim's fixture limits.
