Wasthon — Pillow smoke test

Imports Pillow 11.0.0 — its C core (_imaging: 4.5k lines + 68 libImaging files) compiled to wasm as a standalone bundle (nppil, ~0.4 MB, pure C / hand-written CPython C-API, no Cython/pybind11/Fortran). PNG only: ZipDecode/ZipEncode drive zlib directly. Exercises the four static PyTypeObjects (ImagingCore / PixelAccess / ImagingDraw) and a real PNG round-trip.
