wasthon — scipy suite dashboard

SciPy 1.14.1's own test suites for the three Fortran-free subpackages ported to wasthon — scipy.ndimage (compiled _nd_image/ _ni_label), scipy.special (cephes/xsf/Boost.Math, nine extensions, dstevr on lapack_lite), and scipy.fft (pypocketfft over pybind11). Each module runs in a fresh page against its own wasm bundle (npnd / npsp / npfft) so nothing contaminates anything else. Scores are scipy's own assertions. Only the pytest-portable subset is listed — modules needing scipy.integrate (Fortran), mpmath/hypothesis, threading, the uarray backend registry, or numpy.random are out of scope.
