Wasthon — scipy.cluster test module

Runs a single SciPy 1.14.1 scipy/special/tests/test_*.py module against wasthon — the nine Fortran-free scipy.special extensions (cephes/xsf/Boost.Math, codegen'd ufuncs) compiled to wasm, with the one LAPACK routine special needs (dstevr) emulated on numpy's lapack_lite. Pass ?test=scipy.special.tests.test_basic to pick a module. The dashboard (test-scipy-special-all) drives this page one module at a time.
