wasthon — scipy.fft suite dashboard

Runs modules from SciPy 1.14.1's own scipy.fft test suite (scipy/fft/tests/test_*.py) against wasthon — the pypocketfft pybind11 extension over the header-only pocketfft, with scipy.fft's public API bound to its direct-pocketfft backend (uarray-free) — each in a fresh page (test-scipy-fft.html?test=…). Scores are scipy's own assertions. Note: numpy.random's Generator C-module doesn't init on this bundle, so the modules that build random test arrays (test_basic / test_real_transforms / test_multithreading) import-fail — a numpy.random plumbing gap, not an fft one.
