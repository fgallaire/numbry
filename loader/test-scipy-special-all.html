wasthon — scipy.special suite dashboard

Runs modules from SciPy 1.14.1's own scipy.special test suite (scipy/special/tests/test_*.py) against wasthon — the nine Fortran-free extensions (cephes/xsf/Boost.Math, codegen'd ufuncs), the single LAPACK routine dstevr emulated on numpy's lapack_lite — each in a fresh page (test-scipy-special.html?test=…) so no module contaminates another. Scores are scipy's own assertions. Modules needing mpmath/hypothesis or the *.npz reference vectors (not packaged) are out of scope.
