Wasthon — seaborn smoke test

Imports seaborn 0.13.2 on the full stack — numpy C core + lapack_lite, pandas' 43 compiled extensions, matplotlib's Agg C++ renderer, FreeType and kiwisolver, all in ONE wasm module — draws lineplot / histplot / kdeplot / regplot and blits buffer_rgba() onto the canvas below. The kdeplot and regplot panels run real np.linalg (cholesky/inv/det, pinv) in the browser.
